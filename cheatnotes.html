Q.Why the Term HyperText & Markup Language?

        
        
 term 'Hypertext Markup Language' is composed of two main words: 'hypertext' and 'markup language.'

             'Hypertext' refers to the linking of text with other documents, while 'markup language' denotes a language that utilizes a specific set of tags.
        

        
Thus, HTML is the practice of displaying text, graphics, audio, video etc. in a certain way using special tags.
        

        
Note: Tags are meaningful texts enclosed in angle braces, like '<...>'. For example, the <head> tag. Each tag has a unique meaning and significance in building an HTML page, and it can influence the web page in various ways.

History of HTML:


In 1989, Tim Berners-Lee established the World Wide Web (www), and in 1991, he created the first version of HTML.
From 1995 to 1997, further work was done to develop and refine different versions of HTML.
In 1999, a committee was organized that standardized HTML 4.0, a version still used by many today.
The latest and most stable version of HTML is 5, also known as HTML5.
Feel free to read more history of HTML here on the HTML Wikipedia page but I will move ahead and cover important aspects of HTML.

Basic Structure

            
An HTML document consists of the following structure:

              <!DOCTYPE html>

                 <html>

                <head>

                     <title>My Webpage</title>

                        </head>

                            <body>

                                 <h1>Hello World</h1>

                    </body>

                         </html>

                        
                             
HTML Elements

            
Elements define the structure of an HTML page. Example:

            
            <p>This is a paragraph</p>
             <a href="https://example.com">Click here</a>
             <img src="image.jpg" alt="Description">

Headings

HTML provides six levels of headings, from <h1> to <h6>:

            <h1>Main Heading</h1>
            <h2>Subheading</h2>
            <h3>Smaller Subheading</h3>

Paragraphs and Line Breaks

Paragraphs are created using the <p> tag. Use <br> for line breaks.

            <p>This is a paragraph.</p>
            This is a line break <br> Here is the next line.

Links and Anchors

Use the <a> tag to create hyperlinks.

            <a href="https://example.com">Visit Example</a>

Images

Use the <img> tag to insert images:

            <img src="image.jpg" alt="An example image">

Lists

HTML supports ordered and unordered lists:

            <ul>
                <li>Item 1</li>
                <li>Item 2</li>
            </ul>
        
            <ol>
                <li>First</li>
                <li>Second</li>
            </ol>

Tables

Tables are created using <table>, <tr>, and <td> tags:

            <table border="1">
                <tr>
                    <th>Name</th>
                    <th>Age</th>
                </tr>
                <tr>
                    <td>Alice</td>
                    <td>25</td>
                </tr>
            </table>

Forms and Input

Forms allow user input with various fields:

            <form action="submit.php" method="POST">
                <label>Name: <input type="text" name="name"></label>
                <button type="submit">Submit</button>
            </form>

Div and Span

<div> is a block container, while <span> is inline:

            <div style="background:lightgray">Block container</div>
            <span style="color:red">Inline text</span>

Semantic HTML

Use semantic tags for better readability:

            <header>Page Header</header>
            <nav>Navigation</nav>
            <article>Main Content</article>
            <footer>Footer Content</footer>

Meta Tags

Meta tags provide metadata:

            <meta name="description" content="Best HTML Guide">

HTML5 Features

New elements include <audio>, <video>, and <canvas>:

            <video controls>
                <source src="video.mp4" type="video/mp4">
            </video>

HTML Entities

Special characters can be represented using entities:

            &lt; = <  (Less Than)
            &gt; = >  (Greater Than)
            &amp; = &  (Ampersand)

Iframe

Embeds another webpage inside a page:

            <iframe src="https://example.com"></iframe>

SVG and Canvas

SVG and Canvas are used for graphics:

            <canvas width="200" height="100"></canvas>

Accessibility

Use ARIA attributes and alt text for accessibility:

            <button aria-label="Submit form">Submit</button>

What is CSS?

CSS stands for Cascading Style Sheets. It is a stylesheet language that is used to describe the visual presentation of a web page written in HTML (Hypertext Markup Language).

HTML creates the structure of the page, while CSS adds styling to that structure. This tutorial assumes that you have prior knowledge of HTML. If that's not the case, you can follow the HTML tutorial first.

Why the word "cascade"? The term "cascade" refers to the priority scheme determining which CSS rules are applied when multiple rules target an element. This scheme takes into account specificity and inheritance, ensuring that the most specific and latest rules are applied.

Why use CSS? CSS is used to provide styling to HTML elements, making web pages visually appealing and user-friendly. Consider the video below to understand how CSS can transform a webpage:
